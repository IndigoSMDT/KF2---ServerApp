Startup parametrs:
"-console" - start app with console
"-start" - start server with app launch
"-firststart" - instal server with app launch

Console comands:
Server.Start - start server
Server.Stop - stop server
Server.Update - update server
Server.Install - Install server
Server.FirstLaunch - Imitate first launch server
Update.Info - Load all info about server from ini files
Save.Info - Save all info 
All.Active - Set all button to active state
All.Inactive - Set all button to inactive state
DownloadSteamCMD- Download SteamCMD.exe file
Exit - Close app
